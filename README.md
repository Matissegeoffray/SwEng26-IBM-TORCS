# TCD SwEng26 IBM TORCS
This code is part of a group project ran by Trinity College Dublin and IBM to develop an AI race car driver in TORCS. This repo serves as a place for public releases as we must work on the project using the school's private GitLab server.

## How To Run
```bash
cd gym_torcs/ml
pip install -r requirements.txt
python evaluate.py
```
When TORCS is launching make sure it is in focus as we are simulating keyboard inputs to start the race.

Also note that this ML driver is optimized for speed and is not yet fully stable. It will not always perform a successful lap.
